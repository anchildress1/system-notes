# System Notes // Web

The visual cortex of the System Notes ecosystem. Built to look good while doing heavy lifting.

## Tech Stack

- **Next.js 16** (Bleeding edge because why not)
- **React 19**
- **PixiJS** (For the glitter, obviously)
- **TailwindCSS v4**

## Critical Commands

```bash
# Wake up the UI
npm run dev

# Run the test suite (now with coverage!)
npm run test

# Check if you broke the rules
npm run lint
```
